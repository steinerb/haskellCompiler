# haskellCompiler
A compiler built in haskell

INTSALL HASKELL: (linux)
  $ sudo apt install ghc

INSTALL LIBRARIES
	cabal install
	cabal install Parsec
	cabal install Hex

TO COMPILE:
  $ ghc main.hs
  
TO RUN THE MAIN PROGRAM:
  $ ./main

TO RUN THE TEST OR MAIN PROGRAM:
  new tests will be added soon!
